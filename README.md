### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/ChristianHesels/Home-Server.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the Next.JS app
   ```sh
   npm run build
   ```
4. Start the Server _this will start the application on http://localhost:3000_
   ```sh
   npm run start
   ```
